**File 1: `src/app/api/agents/[id]/sessions/route.ts`**

GET handler:
1. `await params` to get `id` (agentId).
2. Call `requireOwnership(agentId)` â€” return the Response if it is one (401/403).
3. Query `chat_sessions` WHERE `agent_id = agentId`, ORDER BY `last_message_at DESC`, LIMIT 50. Select `id, title, created_at, last_message_at`.
4. Also check for orphaned messages: query `chat_messages` WHERE `agent_id = agentId AND session_id IS NULL`, count only. If count > 0, prepend a synthetic legacy session object to the array: `{ id: '__legacy__', title: 'Previous messages', created_at: <earliest orphaned message created_at>, last_message_at: <latest orphaned message created_at> }`. To get earliest/latest, do a single query: `.select('created_at').eq('agent_id', agentId).is('session_id', null).order('created_at', { ascending: true }).limit(1)` for earliest, and `.order('created_at', { ascending: false }).limit(1)` for latest.
5. Return JSON array.

POST handler:
1. `await params` to get `id` (agentId).
2. Call `requireOwnership(agentId)`.
3. Insert into `chat_sessions` with `{ agent_id: agentId }`. Use `.select().single()` to return the created row.
4. Return the created session as JSON.

**File 2: `src/app/api/agents/[id]/sessions/[sessionId]/route.ts`**

GET handler:
1. `await params` to get `id` (agentId) and `sessionId`.
2. Call `requireOwnership(agentId)`.
3. If `sessionId === '__legacy__'`: query `chat_messages` WHERE `agent_id = agentId AND session_id IS NULL`, ORDER BY `created_at ASC`.
4. Otherwise: query `chat_messages` WHERE `agent_id = agentId AND session_id = sessionId`, ORDER BY `created_at ASC`.
5. Return JSON array.

Both files import `requireOwnership` from `@/lib/auth/guard` and `supabaseAdmin` from `@/lib/supabase/admin`. Use the same params typing pattern as the existing chat route: `{ params }: { params: Promise<{ id: string }> }` (and `{ id: string; sessionId: string }` for the nested route).
