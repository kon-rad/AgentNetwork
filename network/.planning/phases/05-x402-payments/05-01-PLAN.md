       - Import x402ResourceServer, HTTPFacilitatorClient from "@x402/core/server"
       - Import registerExactEvmScheme from "@x402/evm/exact/server"
       - Create facilitatorClient with url "https://x402.org/facilitator"
       - Create and export `server` instance (x402ResourceServer with facilitatorClient)
       - Call registerExactEvmScheme(server)
       - Mark with 'server-only' import

    3. Create `src/lib/x402/client.ts`:
       - Import wrapFetchWithPaymentFromConfig from "@x402/fetch"
       - Import privateKeyToAccount from "viem/accounts"
       - Import ExactEvmScheme from "@x402/evm"
       - Export `createPayingFetch(privateKey: \`0x${string}\`)` function
       - Use privateKeyToAccount to create account from key
       - Return wrapFetchWithPaymentFromConfig(fetch, { schemes: [{ network: "eip155:*", client: new ExactEvmScheme(account) }] })
       - Mark with 'server-only' import (private keys must never reach client)

    4. Create `src/lib/chain/usdc.ts`:
       - Import createWalletClient, createPublicClient, http, parseUnits from "viem"
       - Import baseSepolia from "viem/chains"
       - Import privateKeyToAccount from "viem/accounts"
       - Import erc20Abi from "viem"
       - Export const USDC_ADDRESS = "0x036CbD53842c5426634e7929541eC2318f3dCF7e" (Base Sepolia)
       - Export const USDC_DECIMALS = 6
       - Export async function transferUsdc(toAddress: \`0x${string}\`, amount: string): Promise<\`0x${string}\`>
       - Inside: create account from process.env.BOUNTY_PAYER_PRIVATE_KEY
       - Create walletClient and publicClient for baseSepolia
       - Use publicClient.simulateContract to validate before sending (erc20Abi, "transfer", [toAddress, parseUnits(amount, USDC_DECIMALS)])
       - Call walletClient.writeContract(request) and return txHash
       - Mark with 'server-only' import
       - Follow same pattern as src/lib/chain/erc8004.ts for client creation

    IMPORTANT: Read `node_modules/next/dist/docs/` for any proxy.ts conventions before writing code. Use @x402/next (NOT x402-next). USDC has 6 decimals, not 18.
  </action>
