Implement the server-side payment verification and subscription management API routes.

Purpose: The frontend submits a tx hash after the user's wallet sends 100 USDC. This API verifies the transaction on-chain using viem, confirms the correct amount and recipient, writes the subscription to Supabase, and registers the agent group with NanoClaw so the agent is ready to receive messages.
Output: Two API routes — POST /api/subscriptions (verify + create) and GET /api/subscriptions/[agentId] (status lookup)

Create src/app/api/subscriptions/route.ts as a Next.js Route Handler:

**Request body:** `{ tx_hash: string, agent_id: string }`

**Auth:** Require authentication via `requireAuth()` from `@/lib/auth/guard`. Return 401 if not authenticated.

**On-chain verification using viem:**
```typescript
import { createPublicClient, http, erc20Abi, parseUnits, decodeEventLog } from 'viem'
import { base } from 'viem/chains'

const USDC_CONTRACT = '0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913' as const
const USDC_DECIMALS = 6
const REQUIRED_AMOUNT = parseUnits('100', USDC_DECIMALS) // 100_000_000n

const publicClient = createPublicClient({ chain: base, transport: http() })
```

Verification steps (return 422 if any fail):
1. `publicClient.getTransactionReceipt({ hash: tx_hash })` — confirm receipt exists and `status === 'success'`
2. Find the Transfer log in `receipt.logs` where `address === USDC_CONTRACT` (case-insensitive)
3. `decodeEventLog({ abi: erc20Abi, data, topics })` on that log — confirms it's a Transfer event
4. Confirm decoded `args.to.toLowerCase() === process.env.TREASURY_ADDRESS!.toLowerCase()`
5. Confirm decoded `args.value >= REQUIRED_AMOUNT` (allow overpayment, not underpayment)
6. Confirm `receipt.from.toLowerCase() === session.address.toLowerCase()` (sender matches session wallet)

**Duplicate check:** Before insert, query `supabaseAdmin.from('subscriptions').select('id').eq('tx_hash', tx_hash).maybeSingle()`. If row exists, return 409 `{ error: 'Transaction already used' }`.

**Insert subscription:**
```typescript
const activatedAt = new Date()
const expiresAt = new Date(activatedAt.getTime() + 30 * 24 * 60 * 60 * 1000)

await supabaseAdmin.from('subscriptions').insert({
  owner_wallet: session.address,
  agent_id,
  tx_hash,
  amount_usdc: Number(formatUnits(decodedValue, USDC_DECIMALS)),
  activated_at: activatedAt.toISOString(),
  expires_at: expiresAt.toISOString(),
  status: 'active',
})
```

**NanoClaw registration (fire-and-forget, do not fail the request if NanoClaw is unreachable):**
```typescript
try {
  await fetch(`${process.env.NANOCLAW_URL}/register-group`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-shared-secret': process.env.NANOCLAW_SHARED_SECRET!,
    },
    body: JSON.stringify({
      name: agent_id,
      folder: agent_id,
      trigger: `@agent-${agent_id}`,
    }),
  })
} catch {
  // NanoClaw registration failure is non-fatal — subscription is still recorded
  console.error('NanoClaw registration failed — agent group not registered')
}
```

Return 201 with the created subscription row on success.

Import `formatUnits` from `viem` for converting BigInt to decimal for storage.

const { data } = await supabaseAdmin
