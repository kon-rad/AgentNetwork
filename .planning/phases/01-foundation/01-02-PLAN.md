  <files>

  <action>
1. Update `src/components/layout/navbar.tsx`:
   - Remove the placeholder `<button>Connect</button>`.
   - Import `ConnectButton` from `@rainbow-me/rainbowkit`.
   - Replace the button with `<ConnectButton showBalance={false} chainStatus="icon" accountStatus="avatar" />`.
   - `showBalance={false}` keeps the button compact; `chainStatus="icon"` shows the chain icon (which also serves as the network switcher — satisfies WALL-03); `accountStatus="avatar"` shows the ENS avatar or address.
   - Keep the existing nav structure, logo, and nav items unchanged.
   - Keep the wrapper div around the ConnectButton for layout alignment, but remove the fixed width `w-[100px]` since ConnectButton has its own sizing.

2. Create `src/lib/hooks/use-display-name.ts`:
   - Import `useEnsName` from `wagmi`.
   - Import `mainnet` from `viem/chains`.
   - Export function `useDisplayName(address?: string)` that:
     a. Calls `useEnsName({ address: address as \`0x${string}\` | undefined, chainId: mainnet.id })` — MUST pass `chainId: mainnet.id` (chain 1) because ENS registry lives on Ethereum mainnet, NOT on Base. Without this, all lookups return null when wallet is connected to Base.
     b. If `!address`, returns `{ displayName: null, isEns: false }`.
     c. If `ensName` is available (not null/undefined), returns `{ displayName: ensName, isEns: true }`.
     d. Otherwise, returns `{ displayName: \`${address.slice(0, 6)}...${address.slice(-4)}\`, isEns: false }`.
   - Also export a pure utility function `truncateAddress(address: string): string` that returns `${address.slice(0, 6)}...${address.slice(-4)}` — for use in server components or non-hook contexts.
  </action>
  <verify>
Run `grep "ConnectButton" src/components/layout/navbar.tsx` — should match.
Run `grep "chainId.*mainnet" src/lib/hooks/use-display-name.ts` — should match, confirming ENS queries go to chain 1.
Run `grep "placeholder.*Connect\|>Connect<" src/components/layout/navbar.tsx` — should NOT match (placeholder removed).
  </verify>
  <done>Navbar shows RainbowKit ConnectButton with chain switcher icon. useDisplayName hook resolves ENS names via chainId:1 with truncated hex fallback.</done>
</task>
