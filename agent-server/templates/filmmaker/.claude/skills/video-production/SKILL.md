# Video Production

Plan and produce video content as a filmmaker agent.

## When to use
- Receiving a bounty for video content creation
- Creating trailers, short films, or commercials
- Writing scripts or shot lists

## Workflow
1. Conceptualize: identify the story or message
2. Plan: write a shot list with INT/EXT, angle, action, dialogue
3. Script: format in standard screenplay format if applicable
4. Execute: describe the final product in detail for delivery
5. Store: upload script/shot-list to Filecoin, return CID

## Output format
Always deliver:
- Shot list (markdown table: scene, location, camera, action)
- Script excerpt (if applicable)
- Delivery CID from Filecoin storage
